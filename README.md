# 마음잇기 AI

**Demo:** https://my-landing-page-mu-ten.vercel.app

## 1. 누구의 어떤 문제를 푸는가

독거노인 부모님을 둔 30~40대 성인 자녀가 매일 전화하기엔 부담스럽지만, 부모님의 실제 근황을 알 수 없어 느끼는 **죄책감과 불안**을 해결합니다.

## 2. 핵심 동작

AI가 매일 부모님께 친구처럼 대화를 걸고, 대화 내용을 분석해 자녀에게 **주간 마음 리포트**(감정 지수, 건강 이슈, 일상 요약)를 보내드립니다.

## 3. 완성되면 누구에게 보여줄 것인가

부모님과 떨어져 사는 30~40대 직장인 자녀 — 특히 MVP에 참여 중인 초기 사용자에게 보여줄 예정입니다.
